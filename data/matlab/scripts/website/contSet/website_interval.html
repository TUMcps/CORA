Interval

In CORA, interval are instantiated by
% I = interval(lb,ub);
where lb is the lower bound and ub is the upper bound.
Example:
% initialize interval
lb = [-1;1];
ub = [2;3];
I = interval(lb,ub);
 
% plot interval
plot(I);
For more information, type
help interval
