Zonotope

In CORA, zontopes are instantiated by
% Z = zonotope(c,G);
where c is the center vector and G is the generator matrix.
Example:
% initialize zonotope
c = [1;2];
G = [1 0 1; 0 1 1];
Z = zonotope(c,G);
 
% plot zonotope
plot(Z);
For more information, type
help zonotope
