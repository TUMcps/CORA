Tutorial: How to create a zonotope

In this tutorial, you learn how to create a zonotope [1]
% zonotope
c = [1;2];
G = [1 0 1; 0 1 1];
 
Z = zonotope(c, G);
 
figure; hold on; box on;
plot(Z)

References

[1] Girard, Antoine. "Reachability of uncertain linear systems using zonotopes." International Workshop on Hybrid Systems: Computation and Control. Berlin, Heidelberg: Springer Berlin Heidelberg, 2005.
