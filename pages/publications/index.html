Publications

This pages gives a brief overview of the publications of CORA. We first state the core publications of CORA and afterwards some publications using CORA.

To cite CORA, please use the following reference:

  @inproceedings{Althoff2015ARCH,
	author			= {Matthias Althoff},
	title			= {An Introduction to {CORA} 2015},
	year			= {2015},
	month 			= {December},
	booktitle		= {Proc. of the 1st and 2nd Workshop on Applied Verification for Continuous and Hybrid Systems},
	doi			= {10.29007/zbkv},
	url			= {https://easychair.org/publications/paper/xMm}
	publisher 		= {EasyChair},
	pages			= {120-151}
  }
