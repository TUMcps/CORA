Archive

The first lines of CORA were written in September 2006 by Matthias Althoff. This pages archives all previous versions of CORA and shows the history of CORA.

Please check the  GitHub page for the current version. CORA is improved continuously throughout the year (see the release notes below), so the best practice is to clone the GitHub repository and regularly check for updates.
